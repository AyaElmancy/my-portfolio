Project 9:Gold Purchase Receipt Calculator

This web page prompts the user for their name, the number of grams of gold they wish to buy, and the price per gram. It then calculates the total price for the purchase by multiplying the number of grams by the price per gram and displays the result as a receipt. The receipt includes the user's name and the total cost of the gold purchase, formatted to two decimal places. This simple tool allows users to quickly compute their gold purchase cost based on dynamic inputs.

Project 10:String Analysis and Circle Area Calculator

This web page performs three tasks: It counts the number of occurrences of the letter 'e' in a user-provided string. The page uses simple JavaScript functions to handle these tasks and provides relevant alerts for each action.

Project 11:Basic Arithmetic Operations with User Input

The results are displayed in an alert box with each operation clearly listed. The input validation ensures that the user enters valid numeric values for the calculations.
